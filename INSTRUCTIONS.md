# Vox (内容生成Agent) - 初始指令

> 代号 Vox (声)，意为"声音、音频内容生成"

## 项目定位

**核心问题**：需要批量生成内容（小红书、抖音、公众号），但人工创作太慢，且质量不稳定。

**解决方案**：做一个**垂直领域的内容生成Agent**，参考 MoneyPrinterTurbo（通用视频生成），但专注**文案+配图**的组合输出。

> 定位：**营销内容生成**（文案+图片+适配多平台）
> MVP阶段：先做**文案生成**，配图后续看需求

**阶段规划**：
- 阶段1（当前）：多平台文案生成（小红书、抖音、公众号）
- 阶段2（后续）：配图生成/推荐（DALL-E/Midjourney/素材库）
- 阶段3（后续）：短视频脚本（口播文案+分镜）

## 参考项目分析

### MoneyPrinterTurbo
**Stars**: 52k | **架构**: Python + WebUI + API

**核心功能**:
- 输入主题 → 自动生成视频
- 视频素材来源：Pexels（无版权）
- 支持中文/英文文案
- 支持字幕、背景音乐

**技术要点**:
```python
# 核心流程
1. LLM生成文案 → 2. 获取视频素材 → 3. 合成字幕 → 4. 添加背景音乐 → 5. 输出视频
```

**启发**:
- 完整的视频生成流程
- 素材管理
- 多语音支持

**我们的差异**:
- 我们不做视频，做**文案+图片**
- 专注**国内平台适配**（小红书、公众号、抖音、微博）
- 更短的生成周期（秒级输出）
- **国内市场特色**：中文语境emoji使用、微博热搜结合、微信生态打通

**开源差异** (如果未来发布 GitHub)：
- 原项目 MoneyPrinterTurbo 面向全球，目标是视频生成
- 我们是**营销文案+配图**的轻量工具，专注转化
- 国内平台（小红书、抖音）是原生支持，不只是翻译

**前端设计规范** ⚠️ 必须遵循，避免 AI 死板风格：

1. **动画优先**
   - 必须使用 `motion` (Framer Motion) 做动画
   - 禁止出现"死板"、没有过渡效果的 UI
   - 文案生成、平台切换、配图加载都要有动画
   - 参考：https://motion.dev/

2. **配色方案**
   - ❌ 禁止用默认的 AI 蓝/紫色（#007AFF, #6366F1 等）
   - ✅ 使用成熟的配色方案，根据内容属性自选：
     - 活力感：橙/珊瑚色系
     - 专业感：深灰/藏青 + 金色
   - 参考 shadcn/ui 的默认配色，但可自定义
   - 配色工具：https://ui.shadcn.com/themes
   - MVP阶段：shadcn/ui `zinc` 或 `orange` 主题（活力橙色系）
   - 后续可调整：年度流行色、中国古典色、平台主题色

3. **字体选择**
   - ✅ 使用 Inter 或其他成熟无衬线字体
   - ❌ 禁止用系统默认中文字体
   - 字体平台：Google Fonts

4. **技术栈**
   - Next.js 15 + React 19 + TypeScript
   - Tailwind CSS v4 + shadcn/ui
   - Zustand 状态管理
   - SSE 流式输出（用于文案生成进度）
   - Motion 动画

5. **组件规范**
   - 优先使用 shadcn/ui 组件
   - 组件要有 hover、focus、active 状态
   - Tab 切换要有滑动动画
   - 文案展示要有打字机效果

6. **设计参考**
   - https://v0.dev/ - Vercel AI UI 参考
   - https://ui.shadcn.com/ - shadcn/ui 组件库
   - https://motion.dev/ - 动画库
